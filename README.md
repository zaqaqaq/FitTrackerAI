## ✨ Возможности

### 🔐 Авторизация
- JWT-аутентификация
- Регистрация и вход
- Защищённые маршруты

### 📊 Главная страница
- Статистика тренировок (всего тренировок, подходов, тоннаж)
- Отслеживание личных рекордов
- **16 достижений** с прогресс-барами

### 💪 Тренировки
- Создание, редактирование, удаление тренировок
- Добавление подходов с весом, повторениями и подходами
- Фильтрация по тренировкам

### 📈 Прогресс и аналитика
- **Интерактивные графики** (библиотека Recharts)
- **Калькулятор 1 ПМ** (формулы Бжицки, Эпли, Лэндер)
- **Календарь активности** с отслеживанием серий
- Визуализация прогресса по каждому упражнению

### 🍳 Кулинарный блокнот
- Создание, редактирование, удаление рецептов
- Загрузка изображений с компьютера
- Детальная страница рецепта
- Отслеживание КБЖУ (калории, белки, жиры, углеводы)

### 🤖 AI Тренер
- Умные ответы на основе ключевых слов
- Программы тренировок (новичок, средний, продвинутый)
- Гайды по технике упражнений (приседания, жим лёжа, становая)
- Советы по питанию (на массу / на сушку)
- Советы по восстановлению и мотивация
- **Умные подсказки** после каждого ответа для продолжения диалога

### 📱 Адаптивный дизайн
- Полностью адаптивная вёрстка
- Работает на телефонах, планшетах и компьютерах
- Бургер-меню на мобильных устройствах

## 🛠️ Технологии

### Бэкенд
| Технология | Назначение |
|------------|------------|
| **FastAPI** | Веб-фреймворк на Python |
| **SQLite / PostgreSQL** | База данных |
| **SQLAlchemy** | ORM для работы с БД |
| **JWT** | Аутентификация пользователей |
| **Uvicorn** | ASGI сервер |

### Фронтенд
| Технология | Назначение |
|------------|------------|
| **React 18** + TypeScript | UI библиотека |
| **Tailwind CSS** | Стилизация и адаптив |
| **React Router DOM** | Маршрутизация |
| **Recharts** | Визуализация графиков |
| **Lucide React** | Иконки |

### DevOps
| Технология | Назначение |
|------------|------------|
| **Docker** | Контейнеризация |
| **Docker Compose** | Оркестрация сервисов |
| **GitHub Actions** | CI/CD (готово к настройке) |
